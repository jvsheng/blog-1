背景

聊聊为什么淘宝要提出「双飞翼」布局

讲解的很好，下面是一些自己的理解与实战。

实践

圣杯布局和双飞翼布局解决的问题是一样的，就是两边定宽，中间自适应的三栏布局，中间栏要在放在文档流前面以优先渲染。

双飞翼布局是通过新建的div的外边距隔离各列，而圣杯布局则是通过父容器的内边距来实现各列的间隙。
